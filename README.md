# UW-Green Bay Library — Primo VE NDE Customizations

## Purpose

This repository serves as a backup of our Primo NDE customization package.

## Repository Structure

The bulk of our customizations live in the `assets` folder, as part of the standard Primo NDE customization package structure.

## Using This Code

These customizations are designed for our specific branding and institutional needs, so you'll almost certainly need to adapt them before use.

> **Note:** Ex Libris updates Primo regularly. CSS selectors and class names can change between releases, so some code may become outdated without notice.

## Disclaimer

This repository is primarily maintained as an internal backup. We are not able to provide support or respond to issues.
